== Description ==

Place your slid.es deck inline in your post.  This will create an inline slideshow from Slides.com.

== Installation ==

Using this plugin is very simple. All you have to do is:

1. Upload the `inline-reveal-js` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Add shortcode to your post
